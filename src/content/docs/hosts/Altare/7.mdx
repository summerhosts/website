### Updated economy
- 0.35 credits per minute on AFK page
- 85 credits for you per referral, 350 for the user who signed up
- If your referral code is used in checkout when buying credits, you also earn 40% of it
- 30-95 credits per day (more via streak per usual)
- Invites/boosts unknown right now, invites may stay the same
- Decreased store prices
